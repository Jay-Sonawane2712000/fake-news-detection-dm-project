# Fake News Detection DM Project

This repository contains the final project for CSE 572: Data Mining at Arizona State University.

## Project Overview
This project builds a fake news detection pipeline using the ISOT Fake News Dataset and classical machine learning methods with TF-IDF features.

## Dataset
- `Fake.csv`
- `True.csv`

## Final Model
- Tuned LinearSVC
- Feature source: title + text
- N-gram range: (1,1)

## Final Test Performance
- Accuracy: 0.9916
- F1-score: 0.9923
- ROC-AUC: 0.9990
- Average Precision: 0.9990
